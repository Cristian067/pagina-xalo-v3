4: El AP puede tener 4 modos de funcionamento, ¿cuales son? ¿como actuan cada uno de ellos?

Modo cliente: Se utiliza como receptor y actúa como un cable de red uniéndose a una red.

Modo AP (punto de Acceso): El Punto de acceso sirve de núcleo para la instalación del cableado, de forma que los múltiples usuarios acceden a la red por medio del punto de acceso.

Modo Repetidor: Este modo se puede usar para extender la señal de forma que el punto de acceso amplifica la señal que recibe para optimizar el rango de acción.

Modo Bridge: Este modo se hace para cubrir grandes distancias, como dos edificios separados. Con dos puntos de acceso conectados entre sí podemos conseguir una red WLAN a distancias considerables.


5: ¿Cómo podemos crear SSID nuevos? ¿Qué son los SSID?

Selecciona el ícono de Wireless Setup en el menú principal: 1, si tienes la opción, selecciona "2.4 GHz" o "5 GHz" en la barra lateral izquierda. 2, selecciona "SSID Setup" desde el menú en la barra lateral izquierda. 3, tu red principal debería aparecer primero en la lista desplegable de SSID.

SSID se trata del nombre que tiene cada red Wi-Fi y sirve para identificarla frente a otras que haya dentro del mismo rango. Por ejemplo si hacemos una búsqueda en nuestra vivienda, nos aparecerá el SSID de nuestra red, así como otras de otros vecinos.


Como podemos establecer un sistema de seguridad para credenciales para acceder al SSID?

Deberemos configurar un SSID diferente con una contraseña alternativa a las que tenemos, tanto para el acceso al router como para el acceso a la red wifi local.


Qué modo de seguridad es el más recomendable, WEP, WPA o WPA2? Por qué?

WPA2 porque puedes agregar, editar de una forma más segura y a tu propio estilo de personalización la SSID.


¿Se pueden crear distintas VLAN? ¿Cómo?

Si, con las SSID y se pueden crear hasta 16 VLAN.

Router TL-ER604W

En el Router hay dos puertos WLAN. Para qué sirven? Como se configuran?

Sirven para crear un red de área local inalámbrica y se configuran manualmente desde el router.


1: ¿Como podemos configurar un rango de direcciones dinamicas y otra de estàticas?

Con un servidor DHCP.


2: ¿Para qué sirve la pestaña DHCP Client?

Es una función sirve para comprobar los dispositivos que hay conectados en el router.


3: ¿Y la de DHCP Reservation?

Es una función que le permite al router reservar el alquiler de una dirección IP para el uso de un dispositivo específico en la red, garantizando de manera efectiva que el router no asigna la dirección IP a otros dispositivos de la red.


4: ¿Qué es mirroring? ¿Se puede hacer en el router? ¿Cómo?

Screen mirroring (espejo en pantalla) es una manera de mostrar la pantalla del dispositivo móvil utilizando la tecnología Miracast. Y en el router se puede hacer configurando manualmente un puerto con ajustes específicos de dúplex y velocidad.

6: En la configuración Wifi, como se pueden configurar los SSID?

Manualmente desde el router.


¿Qué es WDS?

Es un sistema de distribución inalámbrica que permite la interconexión inalámbrica de puntos de acceso (AP) en una red.


Como funciona el MAC filtering?

El filtrado MAC funciona dando instrucciones al router para que permita conectarse a los dispositivos cuyo MAC aparezca en un listado.
